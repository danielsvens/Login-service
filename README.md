# Login Service
Generic backend login service

Java 11, Spring boot with postgeSQL database

### Example post to create user

```
{
	"userName": "User12",
	"password": "secret",
	"userDetails": {
		"firstName": "John",
		"lastName": "Doe",
		"email": "example@example.com",
		"role": "admin"
	}
}
```
